Requirement Notes

The project must fulfil the following hardware and software requirements in order to run smoothly on the given interface and provide a better user experience. These requirements may vary with time, i.e. as the data increases more computational power would be required to generate valid predictions from that data
Even software requirements might change with update in modules and libraries, hence keeping version control tool like git would be very helpful in easily migrating those changes. Git even helps with increasing number of collaborators and changes suggested by them, it shows the code changes and hence can be safely committed.
